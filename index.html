What is FURL?

FURL is a simple service written in Node.js that provides an HTTP API for resolving short URLs. You are using it right now!

FURL has been placed into the public domain, so do with it as you wish. If you find it useful, please drop us a note and tell us about it.

The source code is available on the 3ft9 GitHub account.
